# Karo Pitch Demo Website (Intern Assignment)

Demo landing page for **Karo Pitch** by **KaroStartup**.

## Project Objective
Design a modern, clean, investor-focused homepage that explains Karo Pitch and encourages early-stage founders across Bharat to apply.

## Live Demo
- Website Link: [https://bhikhukarmur03.github.io/karo-pitch-landing-page/]
- Homepage Screenshot: `homepage-full.png`

## Assignment Requirements Covered
- Hero section with:
  - “Pitch Your Startup to India’s Top Investors.”
  - Subtext
  - Buttons: **Apply to Pitch**, **Explore Startups**
- About Karo Pitch
- How It Works (4-step flow with icons)
- Who Can Apply categories
- Investors section
- Featured Startups cards (dummy content)
- About KaroStartup
- Final CTA:
  - **Apply Now**
  - **Partner With Us**
- Mobile responsive layout
- Clean navigation + startup-themed visuals/animations

## Design Approach
- Built for a balance of:
  - founder-friendliness
  - investor trust
- Used a clean layout, strong hierarchy, clear CTA flow, and responsive sections.

## AI Tool Used
- **ChatGPT (Codex)** for AI-assisted content structuring, copy refinement, and design workflow support.

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure
- `index.html` — page structure/content
- `styles.css` — styling and responsive design
- `script.js` — reveal animations + mobile nav interactions
- `homepage.png` / `homepage-full.png` — screenshots

## Run Locally
1. Clone/download this repository
2. Open `index.html` in browser
